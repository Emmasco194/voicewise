# **App Name**: VoiceWise AI

## Core Features:

- Text Input: Allow users to input questions via text.
- Voice Input: Enable voice input for users to ask questions using speech.
- AI Chatbot: Utilize the ChatGPT model to generate answers to user queries.
- Text-to-Speech Output: Read out the generated answers using a text-to-speech engine.
- Smart Summarization: AI tool to automatically summarize long responses to concise points.

## Style Guidelines:

- Primary color: Soft blue (#89B4FA) to create a calm and trustworthy atmosphere.
- Background color: Very light blue (#E0F2FE) for a clean and unobtrusive look.
- Accent color: Soft purple (#A699F1) for interactive elements, providing contrast.
- Body and headline font: 'Inter', a sans-serif, providing a clean and modern look for easy readability.
- Use clean and simple icons to represent different functions, such as microphone and text input.
- A clean and intuitive layout with a prominent input area and clear display of responses.
- Subtle animations to indicate loading and processing, enhancing user experience.
